使用流程

1. 创建测试表(示例下载)，测试内容需要写在table > tbody内

  • 测试表格式如下，四列内容分别为：命令、参数1、参数2、描述
    open http://126.am 打开首页
    click_element css=li.st_top_menu_about 5000 点击关于按钮(如果按钮不存在，将重试直到5000ms超时)
    wait 1000 等待动画完成(1000ms)
    save_screenshot about1.png 0x600+0+0 截图(图片高度自定义为600)
  • 每行tr可设置id, if, ifnot属性，示例如下
    <tr id="test1">...  </tr id="">
<tr if="test1">...  </tr id="">      //表示第一条测试通过时才执行此行
<tr ifnot="test1">...  </tr id="">      //表示第一条测试不通过时才执行此行
  • 可配置参数(窗口尺寸、浏览器列表、标题)
    <meta name="window_size" content="1024x780" />
<meta name="browsers" content="chrome/win7,chrome/linux" />
<title>汽车车型对比</title>
  • 其它命令与参数可参考后半部分文档

2. 开始测试：在易测首页上传创建好的表格，测试将自动开启

  • 默认情况下，会同时在ie6/ie7/ie8/ie9/firefox/chrome/opera下执行单元测试

3. 输出报表：

  • 按不同浏览器/操作系统分别输出报表，目前支持的测试环境如下：
    浏览器操作系统
    IE6windows xp

    Chromewindows 7
    ChromeUbuntu 12.10

DOM定位

xpath [wiki]

selenium中默认按xpath定位dom节点，可省略描述开头的"xpath="

例子:

  • /html/body/div[@id='st_main']/div[2]/ul/li[3]/span
  • //img[@alt='The image alt text']
  • //a[contains(@href,'#id1')]
  • //input[@name='name2' and @value='yes']
  • //*[text()="right"]
  • //form[input/@name='username']
  • a[@href='help.php'][name(..)='div'][../@class='header']/@target

id

例子: id=ZeroClipboardMovie_1

css [w3.org]

例子:

  • css=div.popOut
  • css=div.searchcar_types .searchcar_types_flag_0:nth-child(3)

name

根据name定位

例子: name=username

tag_name

根据标签名定位

例子: tag_name=pre

class_name

根据className定位

例子: class_name=popOut

link

例子: link=#123

link_text

例子: link_text=标题

操作函数

open

描述：打开页面

输入：url

  • url(必填): 要打开的页面url

输出：selenium log

window_resize

描述：改变浏览器大小

输入：size

  • size(必填): {width}x{height}，如1280x800

输出：无

wait

描述：等待

输入：wait

  • wait(必填): 等待时间(ms)

输出：无

is_element_present

描述：等待元素出现直到timeout(不作测试)

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：selenium log

move_to_element

描述：将元素滚动到可见视图内，同时鼠标移动到元素上

输入：locator [, offset]

  • locator(必填): DOM定位描述
  • offset(选填): x, y。鼠标相对元素左上角的位移，如果不指定，鼠标将移到元素中心。

输出：selenium log

click_element

描述：点击元素

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：selenium log。如果元素不存在，追加一条is_element_present_ok检查测试

double_click_element

描述：双击元素

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：selenium log。如果元素不存在，追加一条is_element_present_ok检查测试

clear_element

描述：清空元素value

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：selenium log。如果元素不存在，追加一条is_element_present_ok检查测试

type

描述：在当前聚焦元素内输入

输入：lkeystring

  • keystring(必填): 输入字符串，可用"{控制符}+{字符串}"的格式，如control+c, alt+v

输出：selenium log。

send_keys

描述：元素内键盘输入

输入：locator, keystring

  • locator(必填): DOM定位描述
  • keystring(必填): 输入字符串，可用"{控制符}+{字符串}"的格式，如control+c, alt+v

输出：selenium log。如果元素不存在，追加一条is_element_present_ok检查测试

save_screenshot

描述：保存截图

输入：png_name [, size_description]

  • png_name(必填): 输出图片名
  • size_description(选填): 截图范围描述，格式为locator,{width}x{height}+{dx}+{dy}。如果只给定locator，则截取元素所在范围；如果只给定尺寸描述，则按尺寸描述符截取；如果同时给定locator和尺寸描述，则基于元素左上角开始截取。
    save_screenshot full.png 完整页面截图(opera中，不可见部分全黑)
    save_screenshot full.png 1000x600+20+30 在完整截图基础上裁剪1000x600的区域，左上角起始点向右偏移20px，向下30px
    save_screenshot searchcar.png id=searchcar 截取id为searchcar的元素区域
    save_screenshot searchcar2.png id=searchcar,0x450+2+3 截取区域如下：左上角在所选元素左上角基础上位移(2,3)，宽度为元素宽度，高度为450px

输出：selenium log

print_title

描述：输出页面标题

输入：空

输出：页面标题

print_url

描述：输出页面地址

输入：空

输出：页面地址

print_window_size

描述：输出当前窗口大小

输入：空

输出：width: **, height: **

check_alert

描述：检查是否有alert对话框，有则关闭

输入：空

输出：selenium log

refresh

描述：刷新当前页面

输入：空

输出：selenium log

go_back

描述：页面后退

输入：空

输出：selenium log

go_forward

描述：页面前进

输入：空

输出：selenium log

execute_script

描述：执行javascript片断

输入：code

  • code(必填): 要执行的js代码

输出：selenium log

cookie

描述：添加cookie

输入："name=value", "path, domain[, secure]"

  • name=value(必填): cookie名称/值
  • path, domain[, secure](path, domain必填，secure选填): cookie路径/域名/secure(默认为0)
    cookie foo=bar /, 163.com 添加cookie示例

输出：selenium log

输出：selenium log

断言(ok)

is_element_present_ok

描述：判断元素是否存在

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：0(元素不可见)/1(元素可见)

isnt_element_present_ok

描述：判断元素是否不存在

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：0(元素可见)/1(元素不可见)

is_element_displayed_ok

描述：判断元素是否可见

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：0(元素不存在)/1(元素存在)

isnt_element_displayed_ok

描述：判断元素是否不可见

输入：locator [, timeout]

  • locator(必填): DOM定位描述
  • timeout(选填): 等待元素出现的时间(毫秒)

输出：0(元素存在)/1(元素不存在)

hasnt_alert_ok

描述：检查当前有无alert对话框，有则失败

输入：空

输出：selenium log

断言(is|isnt|like|unlike)

  • is断言：判断输入参数是否与期望值相同。
  • isnt断言：判断输入参数是否与期望值不同。
  • like断言：判断输入参数是否满足给定的正则式。
  • unlike断言：判断输入参数是否不满足给定的正则式。

title_*

描述：校验页面title

输入：value

  • value(必填): 标题期望值(is|isnt)或正式表达式(like|unlike)
    title_like .*?_网易汽车论坛 标题为“*_网易汽车论坛”
    title_unlike ^\s*$ 标题不为空

输出：0(页面标题与期望不一致)/1(页面标题与期望一致)

current_url_*

描述：校验页面url

输入：value

  • value(必填): 标题期望值(is|isnt)或正式表达式(like|unlike)
    current_url_like http://club2011.auto.163.com/post/\d+.html 发帖后是否跳转到正确地址

输出：0(页面url与期望不一致)/1(页面url与期望一致)

element_attribute_*

描述：检验元素属性

输入：locator, value

  • locator(必填): DOM定位描述、属性名，中间用逗号","分隔
  • value(必填): 元素属性期望值(is|isnt)或正式表达式(like|unlike)
    element_attribute_is id=test,href http://www.baidu.com 元素属性判断示例

输出：0(元素属性与期望不一致)/1(元素属性与期望一致)

element_css_attribute_*

描述：检验元素css属性

输入："locator, attr", value

  • locator[, attr](必填): DOM定位描述、属性名，中间用逗号","分隔
  • value(必填): 元素属性期望值(is|isnt)或正式表达式(like|unlike)
    element_css_attribute_is id=test,width 200 css属性判断示例

输出：0(元素css属性与期望不一致)/1(元素css属性与期望一致)

辅助函数

  • 可在断言条件的参数中使用辅助函数，语法为{?func(args)?}；还可使用简单的计算表达式，如{?rand(30)+1?}
  • 示例：
    click_element //select[@id='provins']/option[{?rand(15)+1?}] 随机选省份
    send_keys id=birthday {?rand(12)+1980?}-{?rand(12)+1, "%02d"?}-{?rand(30)+1, "%02d"?} 随机输入生日
    send_keys id=qq {?rand(10)+1?}{?randnum(7)?} 随机输入QQ

rand

输入：number

输出：介于0和number之间的整数

randnum

输入：length

输出：长度为length的随机数字串

randaz

输入：length

输出：长度为length的随机小写字母串

randaznum

输入：length

输出：长度为length的随机小写字母/数字串

randstr

输入：length

输出：输出长度为length的随机字符串(0-9, a-z, A-Z, :;<=>?@[\]^_`{|})

randdate

输入：format, fromdate, todate

  • format(选填): 默认为yyyymmdd。可用yyyy或yy表示年份，mm或m表示月份，dd或d表示日期，hh或h表示小时，MM或M表示分钟，ss或s表示秒
  • fromdate(选填): 默认为1970-1-1，可接受的输入格式有：19940203, 1994-02-03, 1994-02-03 14:15:29
  • todate(选填): 默认为当前时间，可接受的输入格式有：19940203, 1994-02-03, 1994-02-03 14:15:29
  • 示例：
    send_keys id=birthday {?randdate('yyyy/mm/dd','19800101','19900101')?} 随机输入生日

输出：按format定义的格式输出介于fromdate和todate之间的随机日期字符串
